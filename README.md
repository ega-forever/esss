# ESSS

Elliptic SSS scheme implementation in Node.js.

[Concept description](musig-esss.v1.pdf) (PDF)

algorithm features
* use secp256k1 to restore secret
* each share can be signed with the corresponding private key


## Installation

```bash
$ npm install
```

## Running test

```bash
$ npm run test
```
